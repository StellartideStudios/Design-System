Button Components

Angular action shells for brand moments, keeping the artwork loud and the behavior native.

Button 1 / Campaign

Hero CTA shell with saturated layering, clipped halftone corner texture, and short display labels.

Use Hero CTAs, launch actions, and event panels.
Label Keep it short so the display type stays punchy.
Focus Aquamarine outline keeps keyboard state visible.
Motion Fast lift and press, disabled for reduced motion.

Button 2 / Signal

A tighter angular button built from the supplied blue, yellow, and red layer artwork.

Shell Blue, yellow, and red source layers compose one button shell.
Class Use stellartide-signal-button, with stellartide-button-2 available as an alias.
Label Short action words fit the flatter, wider silhouette best.
States Hover, press, focus, and disabled mirror Button 1.

Button 3 / Surge

Long banner action shell for high-impact campaign choices and wide event CTAs.

Use Wide campaign CTAs, chapter entries, and feature-forward choices.
Class Use stellartide-surge-button, with stellartide-button-3 as an alias.
Label One short verb keeps the long silhouette crisp.
Shell PNG artwork is decorative; button text remains real HTML.

Button 4 / Tile

Small square action shell redrawn as SVG from the supplied tile, with the original center icon removed.

Use Small campaign actions, compact rewards, inventory affordances, and icon-sized controls.
Class Use stellartide-tile-button, with stellartide-button-4 as an alias.
Shell SVG artwork is intentionally blank in the center so content stays real markup.
Label Keep labels to one or two characters, or pair icon-only use with a clear accessible name.
